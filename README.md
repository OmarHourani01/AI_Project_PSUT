
# AI Project (Heart Attack Risk Classification)

This repo contains a Jupyter notebook that builds and evaluates classification models for **Heart Attack Risk** using `data/heart_attack_prediction_dataset.csv`.

- Main notebook: [heart_attack_classification.ipynb](heart_attack_classification.ipynb)
- Detailed walkthrough: [NOTEBOOK_GUIDE.md](NOTEBOOK_GUIDE.md)

## Quick start

1. Create/activate a Python environment
2. Install dependencies:
	- `pip install -U pandas numpy matplotlib seaborn scikit-learn scipy joblib`
3. Open and run the notebook from top to bottom.
